## 🚀 What I'm currently working on

### [BookNStay](https://github.com/yashwanthkumar-r/bookNstay)

I'm building a hotel booking and management backend with **Spring Boot**. The
project includes:

- Hotel discovery, room management, and inventory tracking
- Guest booking and reservation workflows
- Secure authentication and role-based access with Spring Security and JWT
- Dynamic room pricing using the Strategy Pattern
- PostgreSQL persistence with JPA/Hibernate
- Stripe payments, cancellation refunds, and payment-status tracking

I'm using the project to deepen my knowledge of production-style backend
architecture, API security, design patterns, testing, and reliable workflows.
